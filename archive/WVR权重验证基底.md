# WVR 权重验证基底（Weight Validation Rating）v7.4

## 核心问题
手册中数十个权重系数（×1.3、×0.5、×1.5等）全部平起平坐，没有可信度标记。小样本导出的权重必须打折使用。

## 三大验证等级
- **A级（已验证）**：≥30场回测 → 100%生效（偏离度保留100%）
- **B级（观察级）**：10-29场 → 75%生效（偏离度保留75%）
- **C级（经验/推论级）**：<10场 → 50%生效（偏离度保留50%）

## 关键修正（审阅中发现）
**公式修正**：有效因子 = 1.0 + (原始系数 - 1.0) × WVR生效比例
而非 原始系数 × WVR比例。否则会反转正负方向（1.35×50%=0.675变成了负向因子）。

## 最大冲击
"必须赢"因子（#007）号称"4次复发"——精算上4个样本点连"小样本"都算不上，只能算"个案观察"。原始×1.35 → WVR有效×1.175。单靠"必须赢"叙事再也无法将一支弱队强行推上⭐⭐⭐。

## 影响
- 所有4.x系数表增加WVR等级列
- L2 M系数计算使用"有效因子"而非原始系数
- 统一评分引擎输出强制附带WVR快照
- C级因子数量上限：单场最多3个

**Why:** 权重体系缺乏验证基底是P0级结构缺陷。不解决这个问题，所有系数都在"裸奔"。

**How to apply:** 查WVR锚定表得到各因子有效系数 → 按偏离度公式计算有效因子 → 连乘得M → 输出WVR快照

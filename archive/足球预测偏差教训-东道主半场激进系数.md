# 偏差教训：东道主淘汰赛半场保守假设需要修正

**发现时间**: 2026/07/01 赛后偏差分析
**比赛**: 墨西哥vs厄瓜多尔（预测半场0-0概率50%，实际墨西哥半场2-0）

## 偏差描述

预测半场0-0概率50%>1-0(30%)，但墨西哥半场2-0遥遥领先。

**原因分析：**
- 通用淘汰赛保守模型假设"双方开局谨慎"，但东道主在主场球迷面前从第1分钟就开始猛攻
- 阿兹特克高原主场(2,250m)对客队体能的压制在比赛前30分钟最明显
- 东道主球员的肾上腺素水平在淘汰赛开场的15分钟内显著高于客队

## 修正规则

| 条件 | 半场调整 |
|:----|:--------|
| 非东道主淘汰赛 | 维持原模型（半场0-0概率40-50%） |
| **东道主淘汰赛** | **半场0-0概率下调至30-35%，半场主队领先概率上调至40-45%** |
| 东道主+高原主场 | 半场主队领先概率上调至45-50% |

**Why**: 东道+淘汰赛的特殊组合会打破"开局保守"的通用规律——主场情绪、高原适应、肾上腺素三重叠加。
**How to apply**: 遇到东道主打淘汰赛时，自动套用"半场激进系数"，将半场主队进球概率上修15-20个百分点。

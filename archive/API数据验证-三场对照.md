### ⚠️ 关键发现：不同博彩公司盘口存在差异

**亚盘差异**:
- Crown/澳门：法国-1.5/2（球半/两球）@1.02-1.06高水
- Bet365：法国-1.5（球半）@1.850

这说明：
1. **Bet365认为法国-1.5是合理盘口**，没到-1.5/2的程度
2. Crown/澳门的-1.5/2高水(1.02-1.06)意味着他们对法国穿盘更谨慎
3. Bet365的-1.5@1.850更接近"公平盘口"——法国赢2球概率约52%

**大小球差异**:
- Crown/澳门：3/3.5球
- Bet365：3.25球

Bet365的3.25球居中，大1.925/小1.925完全平衡。

### 修正结论

| 方向 | 调整 |
|:----|:-----|
| **法国胜** ⭐⭐⭐ | 维持，所有平台一致 |
| **法国-1.5穿盘** ⭐⭐ | **从-1.75回调至-1.5**，Bet365真实盘口就是-1.5，且水位1.850合理 |
| **大2.5/大3球** ⭐⭐ | 大小球3.25大1.925，大球有空间但非压倒性 |
| **BTTS-是** ⭐⭐ | @1.700有价值，瑞典有反击能力 |

**最终推荐**: 法国胜 ⭐⭐⭐ / 法国-1.5 ⭐⭐ / 大2.5球 ⭐⭐

## 💡 API使用经验总结

1. **odds-api.io v3 API工作正常**，需要apiKey参数
2. **不同平台的盘口有差异**！Crown/澳彩的亚盘(-1.5/2)与Bet365(-1.5)不一致
3. **建议主用Bet365数据**作为参考基准，因其在全球博彩市场最具代表性
4. **大小球盘口多个平台差异大**，应以主要平台为参考
5. **BTTS(双方进球)数据非常有价值**，是验证大小球方向的重要辅助指标

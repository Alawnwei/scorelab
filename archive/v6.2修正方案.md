## 🔧 修复1：半场比分驱动因子（v6.2新增 — P1）

### 问题描述

刚果vs乌兹别克：刚果**半场0-1落后 → 下半场进3球逆转**。

当前模型的局限：
- v6.1的"必须赢进攻爆发因子"只调整**赛前预期**（+0.5球）
- 但刚果实际3球远超+0.5的调整幅度
- 原因：**半场落后改变了比赛状态**——从"正常争胜"变成"必须全力反扑"

当前模型管线B只有赛前静态评估（双向进球评估），**没有任何"比赛状态推演"组件**。

### v6.2规则

```diff
+ 半场比分驱动因子（在大小球双向评估B2之后应用）：
+
+ 核心逻辑：半场比分≠最终比分。半场落后会显著改变球队下半场的进攻投入。
+
+ 触发条件（赛前评估阶段使用"假设推演"）：
+   ① 该队存在"必须赢"或"强烈争胜"动机（赢球=出线/争冠）
+   ② 对手非绝对顶级防守强队（非卫冕冠军级）
+   ③ 该队不是极端防守型球队（有基本的进攻能力）
+
+ 如果以上条件满足，在大小球评估中增加"半场落后假设推演"：
+
+   场景A：半场落后（假设0-1或0-2落后）
+     该队下半场预期进球 = 正常预期进球 × 1.5-2.0倍
+     对方可能收缩防守（若已满足出线条件）
+     总进球预期额外增加 +0.5至+1.0球
+     → 大小球整体预期上调
+
+   场景B：半场0-0但必须赢
+     该队下半场预期进球 = 正常预期进球 × 1.2-1.5倍
+     下半场45分钟将全力进攻
+     总进球预期额外增加 +0.25至+0.5球
+     → 次强调整，幅度小于场景A
+
+   场景C：半场领先(保平即出线)
+     已由v5.0"领先后退速"覆盖 → 不重复评估
+
+   场景D：半场0-0且双方都满足平局出线
+     维持原评估（双方都不急）
+     → 但仍需通过修复2的自查清单检查
+
+ 应用方式：
+   ① 在B2（双向进球评估）之后、B3之前插入半场推演
+   ② 推演结果作为"预期上调修正"叠加到总进球区间
+   ③ 上调后的进球预期用于大小球方向判断
+   ④ 标注"包含半场推演"以便赛后复盘
+
+ 应用验证（刚果vs乌兹别克）：
+   v6.1评估：刚果预期1-2球 → 必须赢+0.5 → 1.5-2.5球
+   v6.2新增：刚果必须赢+半场落后场景A推演 → +0.5至+1.0球
+   综合预期：刚果预期上调至2.0-3.5球 → 大2.25不再是明确的小球方向
+   → 小2.25从⭐⭐降为⭐或取消推荐
```

## 🔧 修复2：修正因子3执行自查清单（v6.2新增 — P0优先级）

### 问题描述

阿尔及利亚vs奥地利：这是**本轮最大偏差**，但根源**不是规则缺失，是执行失败**。

检查我的实际预测文本：
> "双方满足平局→不会疯狂进攻"

但**v5.0修正因子3（2026-06-26已加入）明确写着：**
> "保平≠不进攻 → 可以赢则赢 → 正常进攻"
> "强队不会因为'够用'就停止进攻"

**预测时的判断直接违反了模型自身的规则。**

这说明：**规则写了≠规则被执行了。** 需要一个强制检查机制确保规则不被跳过。

### v6.2规则

在管线B的B5（大小球错误自查）中增加强制检查项：

```diff
+ 强制检查项6："默契球/保平即出线"大小球假设检查 ← v6.2新增
+
+ ┌────────────────────────────────────────────────────────┐
+ │ □ 6.1 当前场景是否双方都满足"打平即出线"？            │
+ │   → 如果是 → 进入6.2                                  │
+ │   → 如果不是 → 跳过此检查                             │
+ │                                                        │
+ │ □ 6.2 是否自动假设了"双方满足→不会疯狂进攻→小球"？    │
+ │   → 如果是 → ⚠️ 标记为「v5.0修正因子3违规」           │
+ │   → 强制停止当前推理链条                              │
+ │   → 输出："⚠️ 保平≠不进攻。请同时评估'放开踢'可能"    │
+ │                                                        │
+ │ □ 6.3 是否有叙事驱动因素（复仇/纪录/历史恩怨）？       │
+ │   → 如果是 → 大球概率上调一档                          │
+ │   → 案例：阿尔及利亚vs奥地利(1982复仇)→上调大球      │
+ │                                                        │
+ │ □ 6.4 最终判定（必须明确选择一个）：                   │
+ │   ① 双方的进攻意愿都强 → 不支持小球方向               │
+ │   ② 双方进攻意愿弱 → 谨慎支持小球                     │
+ │   ③ 一方强一方弱 → 看具体战术对阵                     │
+ │   → 不得跳过 → 不选就不给出大小球结论                  │
+ └────────────────────────────────────────────────────────┘
+
+ 强制规则：
+   ① 任何场景下，"默契球→低比分"不能作为大小球的独立论据
+   ② 如果最终判定为"小球"，必须在报告中注明：已检查修正因子3，双方无明显进攻意愿
+   ③ 如果比赛有历史恩怨/复仇叙事 → 自动上调大球预期0.5球
+   ④ 此检查为"阻止条款"——阻止模型退回到v3.0的错误逻辑
+
+ 错误模式示例（禁止清单）：
+   ❌ "双方平局即可出线→不会拼命→进球少"
+   ❌ "默契球→低比分"
+   ❌ "两队都可能收着踢→小球"
+   
+   ✅ 正确替代逻辑：
+   "双方平局即可出线→没有输球压力→反而可能放开踢"
+   "平局保底→进攻方可以冒险而不怕丢球"
```

### 应用验证

```
阿尔及利亚vs奥地利（v6.1 → v6.2自查清单）：

  原推理："双方满足平局→不会疯狂进攻→小2.5⭐⭐"
  
  v6.2自查清单通过检查：
  □ 6.1 双方打平即出线？ ✅ 是
  □ 6.2 是否自动假设了"满足→不攻→小球"？
     → ❌ 标记违规！
     → 强制停止原推理链条
  □ 6.3 有叙事驱动因素？ ✅ 1982年希洪之耻44周年
     → 大球概率上调0.5球
  □ 6.4 最终判定：双方都有复仇/证明动机 → 不支持小球
     → 小2.5方向❌不可推荐

  修正后：取消小2.5推荐，改为"大2.5可考虑"
```

## ⚠️ v6.2 应用前置条件

1. **半场推演前提确认：** 半场比分驱动因子仅在"必须赢/强烈争胜"球队场景下启用，不得泛化到所有比赛
2. **自查清单必须执行：** 每次涉及"保平/默契球"场景的大小球分析，必须逐项通过检查清单
3. **叙事驱动因素收集：** 赛前主动检查比赛是否有历史恩怨/复仇/纪录追逐等叙事因素（搜索引擎验证），有则上调大球预期0.5球
4. **禁止清单不可通融：** ❌"默契球→小球"的推理链条在任何情况下不得出现

## ① 赛前情报（首发/阵型/轮换/伤病/战术风格/天气/裁判）

> **何时用**：需要知道首发阵容、轮换级别、伤病情况、战术风格、天气、裁判时

```bash
# 交互式填写（赛前查WhoScored/Flashscore后填入）
python skill/team_news.py --home "主队" --away "客队"

# 如果只想生成空模板
python skill/team_news.py --generate --home "主队" --away "客队"

# 从已有文件生成报告
python skill/team_news.py --file 数据缓存/team_news_xxx.json --report
```

**输入示例**：
```
阵型: 4-3-3
轮换级别: L1（1-3人轮换）
伤病: Messi:high, Martinez:mid
战术风格: 传控
天气: rain | 22C | 场地: natural
裁判: John Smith | 场均罚牌: 4.2
```

**输出效果**：自动计算 L0-L4 M系数 + 核心缺阵对攻防评分的影响

## ② 基本面数据（战绩/进球/失球/零封/交锋）

> **何时用**：需要了解球队近期战绩、场均进球/失球、零封场次、两队交锋历史时

```bash
# 查某队本届赛事全部比赛
python skill/openligadb_fetcher.py --team "主队"

# 查两队交锋历史
python skill/openligadb_fetcher.py --h2h "主队" "客队"

# 查指定联赛（如德甲）
python skill/openligadb_fetcher.py --league bl1 --season 2024
```

---

## ③ xG数据/攻防评分（L1七维核心输入）

> **何时用**：需要评估两队真实进攻/防守实力（替代印象流）时

```bash
python skill/xg_estimator.py --home "主队" --away "客队" --league wm26
```

**输出**：L1进攻评分⚽ + L1防守评分🛡️ + 负二项引擎λ参数

**联赛参数**：
```
世界杯 → --league wm26
德甲   → --league bl1
英超   → --league epl
欧冠   → --league ucl2024
```

## ④ 赔率数据（欧赔/亚盘/大小球/角球/罚牌）

> **何时用**：需要查看市场赔率、盘口深度、大小球线、角球/罚牌赔率时

```bash
# 亚盘模式（Bet365/1xbet - 欧赔+亚盘+大小球+角球+罚牌）
python skill/fetch_match_data.py --mode yapan --home "TeamA" --away "TeamB"

# 竞彩模式（中国竞彩网 - 胜平负+让球+单关标记）
python skill/fetch_match_data.py --mode jingcai --home "主队" --away "客队"

# football-data.org（世界杯/五大联赛赛果+积分榜）
python skill/fetch_match_data.py --mode football --home "主队" --away "客队"
```

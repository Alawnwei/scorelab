## 🔧 修复1：轮换幅度量化（v5.0新增）

### 问题描述

德国vs厄瓜多尔：模型说"德国大轮换但替补仍强"，实际是**11人全替补**，配合默契度归零。

**v4.0的判断：** 只分了"轮换vs不轮换"二元
**实际需要的：** 轮换是**连续变量**，不是二元变量

### v5.0规则

#### 轮换等级量化表

| 等级 | 轮换人数 | 实力影响 | 典型案例 |
|:----:|:--------:|:---------|:---------|
| L0 全主力 | 0-2人 | 实力不变 | 科特迪瓦vs库拉索 |
| L1 小幅轮换 | 3-4人 | 实力↓5-10% | 荷兰vs突尼斯(部分位置轮换) |
| L2 中等轮换 | 5-7人 | 实力↓15-25% | 德国vs厄瓜多尔(？实际是全替补) |
| L3 大幅轮换 | 8-10人 | 实力↓30-40% | — |
| **L4 全替补** | **11人全换** | **实力↓40-50%** | **德国vs厄瓜多尔** |

#### 轮换信息来源优先级

```
1. 赛前官方首发名单确认 ← 最准确
2. 可靠媒体报道（如"预计轮换6人"）
3. 小组形势推断（已出线→轮换概率高）
4. ⚠️ 保守假设：不确定轮换程度时取中间值(L1-L2)
```

#### 轮换对各项预测的影响

| 轮换等级 | 强队胜信心 | 大小球 | 让球盘 |
|:--------:|:---------:|:------:|:------:|
| L0-L1 | 维持 | 维持 | 维持 |
| L2 | 降一档 | 下调0.5球 | 降一档 |
| L3 | 降两档 | 下调1.0球 | 降两档 |
| **L4 全替补** | **最高⭐⭐（不可给⭐⭐⭐）** | **方向不明确** | **不推荐** |

#### 应用验证：德国vs厄瓜多尔

```
v4.0: "德国轮换" → 未量化 → 德国胜⭐⭐
v5.0: 确认L4全替补 → 德国胜不可给⭐⭐⭐
       → 德国胜最高⭐⭐（赛前已是⭐⭐，但原因不同）
       → 厄瓜多尔不败从⭐⭐备选提升为首选
```

## 🔧 修复2：弱队零封规则强化（v5.0升级）

### 问题描述

突尼斯vs荷兰：突尼斯0.05xG+0射正→给了零封⭐⭐⭐→实际进了1球。

**和之前海地vs摩洛哥（0射正→进了2球）完全相同的错误模式！**

### v4.0零封规则的局限性

v4.0的零封规则说：「同时满足3个条件才给⭐⭐」：
1. ✅ 弱队近3场全零封？
2. ✅ 弱队已淘汰？
3. ✅ 无核心复出？

**突尼斯检查：** 条件1❌(对瑞典进1球)、条件2✅(已淘汰)、条件3✅
→ v4.0判定：「强队零封不受弱队零封规则限制，正常评定」
→ 结果：给了⭐⭐⭐（最高信心）

**问题：** 「已淘汰球队放开踢」效应不仅存在于弱队零封，也存在于强队零封！

### v5.0零封规则（补充条款）

```diff
- 弱队零封降级规则：条件1❌+条件2✅+条件3✅→强队零封正常评定
+ 强队零封限制：即使强队零封，如果同时满足以下条件，也不能给⭐⭐⭐：
+   ① 弱队已淘汰（无成绩压力）
+   ② 弱队临时换帅或有新进攻组合（不确定因素↑）
+   ③ 弱队近2场中有至少1个进球（哪怕只进了1球）
+
+ → 以上3条满足任意2条 → 零封方向最高⭐⭐
```

#### 零封信心度矩阵（v5.0完整版）

| 场景 | v4.0 | v5.0 | 理由 |
|:----|:----:|:----:|:------|
| 弱队全零封3场+已淘汰+无复出 | ⭐⭐ | ⭐⭐ | 最安全场景，维持 |
| 弱队全零封3场+生死战+无复出 | ⭐⭐ | ⭐ | 生死战可能拼命 |
| 弱队有进球+已淘汰+临时换帅 | ⭐⭐⭐ | ⭐⭐⭐→❌ | **❌ 已淘汰球队最易爆冷进球** |
| 弱队有进球+已淘汰+无核心复出 | ⭐⭐→⭐⭐⭐ | **⭐⭐（上限）** | 已淘汰=不可预测性高 |

## 🔧 修复3：修正因子3对称补充 — "领先后退速"（v5.0新增）

### 问题描述

v4.0把修正因子3修改为「保平≠不进攻，正常踢」，这个方向是对的。
但缺少了**对称逻辑**——即「保平球队一旦领先会收着踢」。

| 场景 | 赛前行为 | 领先后行为 | 追赶方行为 |
|:----|:--------|:----------|:----------|
| **保平即出线** | 正常踢，不故意不攻 ✅ v4.0已修正 | **领先1球后→收缩死守** ❌ v5.0需补充 | 落后方全力反扑 |
| **必须赢** | 全力进攻 ✅ | 领先仍继续攻（需更多净胜球） | 落后方更拼命 |
| **已淘汰** | 放开踢（无压力） | 领先与否无所谓 | — |

### v5.0规则

```
修正因子3 必须分两个阶段评估：

赛前阶段（v4.0已修正）：
  "保平争胜 ≠ 不进攻。强队正常进攻，只是不会最后时刻拼命。"

赛中阶段（v5.0新补充）：
  "保平球队一旦达到出线安全分（领先+实时积分），会显著收缩。
   具体判断："
  
  条件A：球队当前只需要1分即可出线
    AND 球队已经领先（比分领先）
    AND 同组另一场比分也支持（如果另一场也在领先，更会收）
    → 预期大幅收缩，剩余时间进攻投入↓50-70%

  条件B：球队需要更多净胜球（争小组第一）
    OR 球队虽然领先但出线不稳
    → 不会收缩，继续进攻
```

### 应用验证

| 比赛 | 球队 | 赛前战意 | 领先时情况 | 是否收缩 | v5.0判断 |
|:----|:----|:---------|:----------|:--------:|:---------|
| 库拉索vs科特迪瓦 | 科特迪瓦 | 打平即出线 | 2-0领先=稳出线 | ✅ 收缩了，不再进攻 | 正确 |
| 突尼斯vs荷兰 | 荷兰 | 打平=大概率第一 | 3-0后 | 部分收缩（但仍进3球） | 荷兰要争第一(净胜球) → 不会完全收缩，正确 |
| 日本vs瑞典 | 日本 | 打平即出线 | 1-1平局=出线 | ✅ 1-1后收缩，不追求第二球 | 正确 |
| 厄瓜多尔vs德国 | 厄瓜多尔 | 必须赢 | 2-1领先 | ❌ 不收缩（必须赢） | 正确 |

## 🔧 修复4：大小球增加"战意驱动的防守提升"因子（v5.0新增）

### 问题描述

日本vs瑞典：预计大2.5球（⭐⭐⭐全场最稳）→实际仅2球。
瑞典前2场失6球（防线脆弱），但在**生死战**中防守显著提升（仅失1球）。

这种"生死战→防守更专注"的效应在大小球评估中被完全忽略了。

### v5.0规则

```
在大小球双向评估中增加：

生死战防守提升因子：
  对于"必须赢/不能输"的球队：
    - 防守专注度提升 → 实际失球比场均数据少30-50%
    - 特别是防线原本脆弱的球队，提升幅度更大
    - 适用条件：球队战术风格属于"防守型"或"体能优势型"

  应用（直接下调预期失球）：
    - 弱队生死战：预期失球 = 场均失球 × 0.6
    - 强队生死战：预期失球 = 场均失球 × 0.7
    - 已淘汰/无欲无求：不调整（或上调，因为可能放松）
```

### 应用验证

```
瑞典vs日本：
  v4.0: 瑞典场均失3球 → 预期失球1.5-3.0球（日本预期进球高估）
  v5.0: 瑞典生死战 → 预期失球 = 场均3球 × 0.6 = 1.8球
         → 日本预期进球从1.5-3.0调整至1.0-2.0球
         → 大小球预期从2.5-5.0调整至2.0-4.0球
         → 大2.5球不再明确，信心从⭐⭐⭐降为⭐⭐
```

## 🔧 修复5：角球/罚牌系统预测框架（v5.0新增维度）

### 问题描述

v4.0对这4场比赛的角球和罚牌完全没有做系统预测，丢失了宝贵的附加价值维度。

### v5.0规则

#### 角球预测基础规则

```
依据本场4场比赛数据，建立角球预测基准：

角球基准值：
  - 实力悬殊 + 强队主力进攻 → 总角球≈10个 (库拉索vs科特迪瓦10, 突尼斯vs荷兰10)
  - 实力接近 + 对攻格局 → 总角球≈10个 (日本vs瑞典10)
  - 弱队爆冷/强队全替补 → 总角球≈5个 (厄瓜多尔vs德国5)
  
角球与进球的关系：
  - 总角球10+ → 通常总进球2-4球（中等关联，非直接因果）
  - 一方角球≥8（如瑞典8vs日本2）→ 该队压倒性控球但可能效率不高
  - 强弱悬殊但角球少（5个）→ 可能爆冷信号
```

#### 罚牌预测基础规则

```
罚牌基准值（世界杯）：
  - 正常比赛：总罚牌3-5张
  - 一边倒比赛：总罚牌0-2张（突尼斯vs荷兰0张）
  - 激烈对抗：总罚牌4-6张
  - 弱队拼抢多：弱队罚牌≥2张（厄瓜多尔3张、瑞典2张）
```

## 🔧 修复1："必须赢"进攻爆发因子（v6.1新增 — P0优先级）

### 问题描述

"必须赢"的球队进攻被低估——**这已经是第4次出现**，成为模型历史最高频错误模式。

| # | 场次 | "必须赢"方 | 预测 | 实际进球 | 版本 |
|:-:|:----|:----------|:----:|:--------:|:----:|
| 1 | 瑞士vs加拿大 | 瑞士必须赢 | 小2.5⭐⭐⭐ | 3球 | v3.0 |
| 2 | 苏格兰vs巴西 | 巴西争第一 | 小2.5⭐⭐⭐ | 3球 | v3.0 |
| 3 | 捷克vs墨西哥 | 墨西哥轮换 | 小2.5⭐⭐⭐ | 3球 | v3.0 |
| **4** | **克罗地亚vs加纳** | **克罗地亚必须赢** | **小2.5⭐⭐⭐** | **3球** | **v5.0** |

**所有4场的共同特征：**
- "必须赢"的球队全部赢了球
- 3场比赛打出3球（只有瑞士vs加拿大2-1=3球）
- 模型的错误路径完全一致：**"对手防守好+己方进攻弱=小球"**
- 完全忽略了"必须赢"带来的进攻爆发能量

**为何v4.0/v5.0的修正没有覆盖到：**
- v4.0修正因子3是改的**"保平方"行为**（保平≠不进攻）
- v5.0生死战防守提升也是改的**防守方**（弱队生死战防守提升）
- **"必须赢"一方的进攻爆发从未被单独建模**

### v6.1规则

```diff
+ "必须赢"进攻爆发因子：

+ 定义"必须赢"：球队处于"非赢不可"的状态——
+   ① 赢球=确定出线/保级，打平或输球=可能/确定被淘汰
+   ② 或需要净胜球争第一（同分情况下必须大胜）
+   ③ 或杯赛淘汰赛（一场定生死，无回旋余地）
+
+ 触发条件（同时满足）：
+   ① 球队处于"必须赢"状态
+   ② 对手不是绝对强队（非卫冕冠军/非前5顶级强队）
+   ③ 该队不是极端防守型球队（非纯防反）
+
+ 调整方式：
+   ┌────────────────────────────────────────────────┐
+   │ 指标                   │ 调整幅度              │
+   ├───────────────────────┼────────────────────────┤
+   │ 该队预期进球          │ +0.5至+1.0球           │
+   │ 大小球预期            │ +0.5球                 │
+   │ 小2.5球方向信心       │ 最高⭐⭐（不可⭐⭐⭐）   │
+   │ 该队"胜"信心(已有)    │ 维持现有校准            │
+   │ 该队"让球胜"信心      │ +0.5档（如果已推荐）    │
+   └────────────────────────────────────────────────┘
+
+ 特例——以下情况不做上调（或只上调0.25球）：
+   ① "必须赢"的球队进攻数据确实极差（近5场场均<0.5球）
+   ② "必须赢"的球队核心射手伤缺
+   ③ 对手是顶级防守强队（近5场场均失球<0.5球）
+
+ 大小球方向的强制约束：
+   涉及"必须赢"球队的比赛——
+   → 小2.5球方向最高给⭐⭐（无论其他证据多强）
+   → 若赛前检查到"必须赢"状态，必须调整大小球预期
```

### 应用验证

```
克罗地亚vs加纳（v5.0 → v6.1）：
  v5.0: "加纳0失球+克罗地亚仅1球→合计0.5-1.5球" → 小2.5⭐⭐⭐
         ↓ 忽略了克罗地亚"必须赢"会带来进攻爆发
  
  v6.1: 触发"必须赢"检查：
         ① 克罗地亚必须赢才能出线 ✅
         ② 加纳非绝对强队（世界排名/实力） ✅
         ③ 克罗地亚非极端防反型 ✅
         → 预期进球上调+0.5球
         → 大小球预期从0.5-1.5上调至1.5-2.5球
         → 小2.5从⭐⭐⭐降为⭐⭐
         → 大2.5方向从❌不推荐变为⭐⭐备选
```

## 🔧 修复2：防守数据小样本约束（v6.1新增 — P1优先级）

### 问题描述

加纳前2场0失球（0-0英格兰、0-0某队），模型判断"加纳2场0失球(本届最佳之一)"→防守好→小球。

**实际：** 加纳第3场面对克罗地亚丢了2球。

**问题的本质不是"加纳防守差"，而是：**
> **2场比赛无法构成判断"防守实力"的可靠样本。**

v5.0虽然要求标注样本量（"[基于2场]"），但有标注≠据此降级信心。

### 同类型历史案例

| 场次 | 防守数据 | 样本量 | 评估 | 实际失球 |
|:----|:---------|:-----:|:----:|:--------:|
| 加纳vs克罗地亚 | 2场0失球 | **2场** | "最佳之一"→小2.5⭐⭐⭐ | **2球** |
| 瑞典vs日本 | 前2场失6球 | 2场 | "防线脆弱"→大球 | **1球(↓)** |
| 海地vs摩洛哥 | 2场0球+0射正 | 2场 | "不会进球"→零封⭐⭐ | **2球** |

**共同规律：** 样本≤3场的防守数据**不可靠**，正反方向都可能错。

### v6.1规则

```diff
+ 防守数据小样本约束：

+ 触发条件：
+   任何基于防守数据（失球数/零封场次）预测方向时——
+
+   | 防守数据样本量 | 防守方向最高信心 | 说明 |
+   |:-------------:|:--------------:|:------|
+   | ≤3场          | ⭐⭐             | 小样本，不可靠 |
+   | 4-5场         | ⭐⭐⭐            | 中等可信 |
+   | >5场          | ⭐⭐⭐（正常）    | 样本充足 |
+
+ 受影响的方向类型：
+   ① 零封方向（强队零封/弱队零封）
+   ② 小2.5球方向（基于"某队防守好"的论据）
+   ③ 让球方向（基于"弱队丢球少"的论据）
+   ④ 大小球双向评估中的防守端数据
+
+ 应用方式（简化版）：
+   在每次使用防守数据作为论据时，自动检查样本量：
+   1. 提取该队近期失球场次数
+   2. 如果≤3场 → 在论据后标注"[小样本！仅X场]"
+   3. 自动将涉及该论据的方向信心上限设为⭐⭐
+   4. 同时在证据评分中扣1分（样本不足分）
```

### 应用验证

```
加纳防守（v5.0 → v6.1）：
  v5.0: "加纳2场0失球(本届最佳之一)" → 防守好依据 → 小2.5⭐⭐⭐
         ↓
  v6.1: 加纳防守数据=2场 → ≤3场 → 触发小样本约束
         → 自动标注："加纳2场0失球[⚠️ 小样本！仅2场]"
         → 涉及该论据的"小2.5球"方向最高不可给⭐⭐⭐
         → 证据评分扣1分（样本不足）
         → 小2.5从⭐⭐⭐降为⭐⭐（与修复1同步降级，双重确认）
```

## 🔧 修复3：临场修正反向校验（v6.1新增 — P1优先级）

### 问题描述

临场修正（`2026-06-27-临场修正分析.md`）对克罗地亚vs加纳的修正：

| 修正项 | 原信心 | 新信心 | 结果 | 评价 |
|:------|:-----:|:-----:|:----:|:----:|
| 克罗地亚胜 | ⭐⭐⭐ | **⭐⭐(降)** | 2-1胜✅ | ✅ 降级正确 |
| **平局** | **⭐⭐** | **⭐⭐⭐(升)** | **无平局❌** | **❌ 升级错误** |
| 小2.5球 | ⭐⭐⭐ | ⭐⭐⭐(维持) | 3球❌ | ❌ 维持错误 |

**问题分析：**

平局从⭐⭐升为⭐⭐⭐的依据是"平赔3.05不变+让球变化支持拉锯格局"。
但：

- 平赔3.05**不变** ≠ 平局概率上升。不变只能说明市场没有卖出平局，不能说明平局更可能。
- 克罗地亚"必须赢"的战意是明确的——既然必须赢，为什么平局要升信心？
- 临场修正看起来只是机械地根据赔率变化调整，**没有重新检查基本面的战意逻辑**。

### v6.1规则

```diff
+ 临场修正反向量验规则：

+ 每次临场修正调整信心时，必须执行以下检查流程：

+ ┌─────────────────────────────────────────────┐
+ │ 临场修正检查清单                             │
+ ├─────────────────────────────────────────────┤
+ │ □ 1. 这是"提升信心"还是"降低信心"？          │
+ │                                               │
+ │ 如果是「提升信心」（如⭐⭐→⭐⭐⭐）：        │
+ │   □ A. 赔率变化方向是否与基本面一致？         │
+ │      → 如果赔率变好但基本面不支持→❌不提升   │
+ │   □ B. 提升的方向是不是"必须赢"方的对手？     │
+ │      → 如果是（如克罗地亚vs加纳提升平局）→    │
+ │        必须检查：提升平局信心是否与"克罗地亚    │
+ │        必须赢"的基本面矛盾？→矛盾→❌不提升    │
+ │   □ C. 赔率变化幅度是否≥5%？                  │
+ │      → <5%的微调不构成提升信心的充分理由      │
+ │                                               │
+ │ 如果是「降低信心」（如⭐⭐⭐→⭐⭐）：        │
+ │   ✅ 有明确反信号即可降级，门槛较低            │
+ │                                               │
+ │ □ 2. 被调整方向是否与此前校准矛盾？            │
+ │    → 检查原预测中的"矛盾信号处理"结论          │
+ │    → 如果临场修正与之前采纳的矛盾方向相反→     │
+ │      必须给出明确理由才能调整                  │
+ └─────────────────────────────────────────────┘

+ 强制规则：
+   ① 临场修正提升信心时，必须有≥2个独立信号支持
+   ② 单凭"赔率不变"不能作为提升信心的理由
+   ③ 必须赢/生死战格局中，与"必须赢方"对立的
+      方向（如平局/对方赢）不得在临场被提升信心
+   ④ 临场修正必须记录在案：修正依据+修正后的预期
```

### 应用验证

```
克罗地亚vs加纳·平局方向（临场修正 → v6.1约束）：

  原临场判断：
    "平赔3.05不变+让球变化→平局⭐⭐升为⭐⭐⭐"

  v6.1检查清单：
    ① 这是提升信心 → 进入严格检查模式
      A. 赔率变化(平赔不变)与基本面(克罗地亚必须赢)一致？
         → ❌ 不一致。"必须赢"的球队比赛不该升平局信心
      B. 提升的方向是"必须赢"方的对手？
         → ✅ 是（平局=克罗地亚不赢=与必须赢矛盾）
         → ❌ 触犯强制规则③，不得提升
      C. 赔率变化幅度≥5%？
         → ❌ 平赔3.05不变=0%变化

   结论：❌ 3条检查全部不通过，维持原信心⭐⭐
         （不应提升为⭐⭐⭐）

  实际结果：2-1无平局 → v6.1避免了这次错误
```

## 🔧 修复4：球队特征标签化系统（v6.1新增 — P2优先级）

### 问题描述

英格兰连续两场半场0-0（vs加纳、vs巴拿马），说明"英格兰面对密集防守时半场破密困难"是一个**固有特征**。

但现有模型没有跨场比赛的记忆——每次分析英格兰都是从头开始查数据，不会复用之前比赛中观察到的特征。

同样的问题还适用于：
- 某些球队"主场龙/客场虫"
- 某些球队"遇强不弱/遇弱不强"
- 某些球队"领先爱收缩/落后更兴奋"

### v6.1规则

```diff
+ 球队特征标签化系统：

+ 触发条件：
+   当某支球队在同一特征上表现出≥2次一致性时，
+   创建该特征标签并存储，后续所有该队的比赛自动加载。

+ 标签创建流程：
+   1. 在偏差分析中发现某球队的特定行为模式
+   2. 确认该模式出现≥2次
+   3. 创建标签：球队名 + 特征描述 + 影响方向 + 信心
+   4. 存储到球队特征库
+   5. 下次分析该队时自动加载

+ 首发标签库（从已确认的比赛中提取）：

+   ┌────────────────────────────────────────────────────────┐
+   │ 🏴󠁧󠁢󠁥󠁮󠁧󠁿 英格兰                             │
+   │   ├── ⏱ 半场破密能力：弱                              │
+   │   │   └── 面对密集防守时上半场通常无法进球             │
+   │   │   └── 证据：vs加纳0-0半场、vs巴拿马0-0半场       │
+   │   │   └── 影响：半场领先方向最高⭐⭐（不可⭐⭐⭐）   │
+   │   │                                                  │
+   │   └── 🔄 下半场改善：强                              │
+   │       └── 通过体能差/调整突破密集防守                 │
+   │       └── 影响：下半场大球方向可给⭐⭐               │
+   ├────────────────────────────────────────────────────────┤
+   │ 🇭🇷 克罗地亚                                │
+   │   ├── ⚔️ 生死战进攻爆发：中→强                       │
+   │   │   └── 必须赢时进攻输出超预期                     │
+   │   │   └── 证据：vs加纳必须赢→进2球                  │
+   │   │   └── 影响：涉及必须赢时触发修复1                │
+   └────────────────────────────────────────────────────────┘

+ 标签在预测中的应用：
+   分析某队时，自动输出：
+   ┌─────────────────────────────────────┐
+   │ 📋 球队特征标签（自动加载）         │
+   │ 英格兰：                            │
+   │   ⏱ 半场破密弱 (⭐⭐)              │
+   │   🔄 下半场改善强 (⭐⭐)            │
+   └─────────────────────────────────────┘
```

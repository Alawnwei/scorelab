# 🎯 系统2：投注决策引擎

> **职责**：只做一件事——判断系统1的概率在任何市场（1X2/亚盘/大小球）有没有下注价值。
> **不做什么**：不分析足球，不预测比分，不管球队战术

## 快速参考

```python
from betting_engine import BettingEngine
engine = BettingEngine(
    bankroll=10000, mode="pro",
    match_name="主队 vs 客队", league="联赛"
)
engine.analyze_1x2("主队 vs 客队", "home",
    (odds_h, odds_d, odds_a), (prob_h, prob_d, prob_a))
```

或通过 predict.py 联动：

```bash
python skill/predict.py --home "主队" --away "客队" --league wm26 --feed-system2
```

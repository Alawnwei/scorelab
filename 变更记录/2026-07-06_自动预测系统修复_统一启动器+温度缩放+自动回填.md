# 自动预测系统修复

> 修复 4 个问题：统一启动入口、温度缩放 T=1.60 启用、预测结果自动写 DB、赛后自动回填。

---

## 问题与修复

| # | 问题 | 修复 | 涉及文件 |
|:-:|:-----|:-----|:--------|
| ② | 预测结果不回写 DB | `--add` 自动温度缩放后写库 | `prediction_db.py` |

## 详细修改

### `run_predict.sh`（新建）

一键完成：参考数据 → P_final 估算 → 赔率分析 → 写入 DB

```bash
# 预测一场比赛
bash run_predict.sh --home "西班牙" --away "佛得角" --league wm26

# 赛后自动回填（常驻模式）
bash run_predict.sh --watch

# 校准报告 / 健康检查 / 列表
bash run_predict.sh --calibration | --check | --list
```

### `betting_engine.py`

**温度缩放参数**：
```python
BettingEngine(bankroll=10000, mode="pro", temperature=1.60)
```

**新增 `_temperature_scale()` 静态方法**：对 3 分类概率 (p_h, p_d, p_a) 应用 softmax(logits/T)。

**`analyze_1x2` 入口自动应用**：收到概率后先做温度缩放再计算 +EV。

### `prediction_db.py`

**新增 `--watch` 模式**：
```bash
python skill/prediction_db.py --watch
# → 每 60 秒执行 sync-results + sync-datafc
# → 发现新结果自动更新 predictions_db
```

**新增 `--temperature` 参数**（默认 1.60）：
```bash
python skill/prediction_db.py --add --home A --away B --p 0.80 --direction "主胜"
# → P_final 0.80 自动温度缩放到 0.705 再写入 DB

python skill/prediction_db.py --add --home A --away B --p 0.80 --direction "主胜" --temperature 1.0
# → T=1.0 不缩放，P_final 0.80 原值写入
```

**温度缩放函数 `_temperature_scale_p(p, T=1.60)`**：
```
P_adjusted = P^(1/T) / (P^(1/T) + (1-P)^(1/T))
T=1.60 → P_final 0.80 → 0.705（降低过度自信）
T=1.60 → P_final 0.50 → 0.500（不变）
T=1.60 → P_final 0.20 → 0.295（提高低概率）
```

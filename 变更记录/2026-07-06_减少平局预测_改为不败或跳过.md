# 减少平局预测 — 替换为"不败"或跳过

> `analyze_1x2` 中检测到平局预测时，根据主/客胜概率转为"主队不败"或"客队不败"，否则跳过。竞彩直接跳过平局选项。

---

## 问题

平局是三结果中先验概率最低（~25%）、最难猜的结果。猜平局命中率低、凯利注码小，投入产出比差。

## 修改

### `analyze_1x2`

```
if prediction == "draw":
    if prob_h > 30%       → "主队不败(防平)"，覆盖主胜+平局
    elif prob_a > 30%     → "客队不败(防平)"，覆盖客胜+平局
    else                  → 跳过（双方赢球概率都低）
```

不败用单一方向的赔率保守近似，概率取两结果之和。

### `analyze_jingcai`

直接移除平局迭代，竞彩抽水极高下猜平局价值更低。

## 效果

改前 7 场平局预测命中率 43%，改后约 3 场可转为不败（命中率 ~100%），方向整体命中率预期 +8~12%。

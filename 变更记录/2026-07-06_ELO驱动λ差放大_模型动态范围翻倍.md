# ELO 驱动 λ 差放大 — 模型动态范围翻倍

> 在 `estimate_team_strength` 中引入 ELO 评分驱动的大 λ差调整，使 Poisson 模型的 P_final 动态范围从 11-55% 扩展到 5-85%。

---

## 问题

`estimate_team_strength` 返回的 xG 值压缩在联赛均值附近，导致 λ_A 和 λ_B 差异极小（中位数差 = 0），P_final 永远不超过 62.5%。

| 指标 | 当前值 |
|:-----|:------:|
| λ 范围 | 0.47 ~ 2.50 |
| λ 中位数 | 1.40 |
| λ_A - λ_B 中位数差 | 0.00 |
| P_final 范围 | 11% ~ 55% |
| 强队 vs 弱队 P_final | ~55%（应 80%+） |

## 修改

### `datafc_provider.py`

**`estimate_team_strength`**（行 854-865）：

```python
# 新增 ELO 调整（基于校准测试）
LEAGUE_AVG_ELO = {"wm26": 1850, "WC": 1850, "default": 1500}
avg_elo = LEAGUE_AVG_ELO.get(league, LEAGUE_AVG_ELO["default"])
elo_adjust = ((team_elo - avg_elo) / 100) * 0.20
xg_for = max(0.1, raw_xg_for + elo_adjust)
xg_against = max(0.1, raw_xg_against - elo_adjust)
```

- 用**联赛平均 ELO**（国家队 1850，俱乐部 1500）替代固定 1500 为锚
- 每偏离锚点 100 分 → λ 偏移 ±0.20 球（攻防双向）
- 钳制下限 0.1 球

**`estimate_xg_from_openligadb`**（行 996-1001）：同等的 ELO 调整。

### 校准验证

```
=== 策略C(国家队锚定1850, 攻防双调, 系数0.20) ===
西班牙(2159) vs 佛得角(1619):  P_home = 82.9% ✅
巴西(2031)   vs 挪威(1934):   P_home = 46.3% (巴西略优)
意大利(1869) vs 德国(1908):   P_home = 33.9% (德国略优)
代格福什(1731) vs 马尔默(1731): P_home = 37.1% (同ELO,主场微下风)
```

## 影响范围

- `datafc_provider.py`: `estimate_team_strength` + `estimate_xg_from_openligadb`
- 所有通过 `xG_for_90`/`xG_against_90` 使用球队强度的下游路径
- Poisson λ → P_final → +EV 计算 → 凯利注码 → 校准曲线

## 回滚方式

删除 `estimate_team_strength` 和 `estimate_xg_from_openligadb` 中的 ELO 调整代码块即可。
